This VS Code extension uses a standard AI architecture. The main entry point `extension.ts` activates on load, setting up the webview UI and commands. The UI is built with HTML/CSS/JS in `webview/`, featuring a chat interface and code editor. The agentic loop processes user messages via the webview, using LangChain for orchestration, a vector database for code context, and Tree-sitter for parsing. The AI provider generates responses that flow back to the UI. Key directories include `src/extension.ts` for activation, `src/webview/` for UI, `src/agent/` for AI logic, `src/database/` for embeddings, `src/parsers/` for analysis, and `src/config/` for settings. The extension offers code explanation, Q&A, and AI-assisted coding.
